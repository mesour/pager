Mesour\Pager

 

Mesour\Pager

Required

Server:

  • PHP >= 5.4
  • Mesour\Components

Client:

  • mesour.components.js

Install

With composer. Go to getcomposer.org.

"require": {
    "mesour\pager": "dev-master"
}

Or go to GIT repository

 

Defaults

$defaults = array(
    self::MAIN => array(
        'el' => 'ul',
        'attributes' => array(
            'class' => 'pagination',
        ),
    ),
    self::ITEMS => array(
        'el' => 'li',
    ),
    self::ITEMS_A => array(
        'el' => 'a',
    ),
    self::WRAPPER => array(
        'el' => 'nav',
        'attributes' => array(),
    )
)
 

Features

Warning: You have to using with Mesour\UI\Application

$application = new \Mesour\UI\Application();

$application->setRequest($_REQUEST);

$application->run();

$pager = new \Mesour\UI\Pager('pager');

$application->addComponent($pager);

$pager->setCounts(55, 10);

$pager->render();
